## Project Overview
This project aims to perform customer segmentation using clustering techniques based on credit card customer behavior data.

The clustering process is used to group customers with similar financial and purchasing patterns.

## Dataset
clusteringmidterm.csv

## Methods Used
- K-Means Clustering
- PCA (Principal Component Analysis)

## Workflow
- Data Loading
- Exploratory Data Analysis (EDA)
- Missing Value Handling
- Outlier Removal
- Feature Scaling
- Elbow Method Analysis
- K-Means Clustering
- Cluster Evaluation
- PCA Visualization
- Correlation Heatmap
- Conclusion

## Evaluation Metrics
- Silhouette Score
- Davies-Bouldin Index (DBI)

## Best Result
- Number of Clusters: 4
- Silhouette Score: 0.248
- Davies-Bouldin Index: 1.438
