## Project Overview
This project aims to build a machine learning model to detect fraudulent online transactions using transaction data.

2. Exploratory Data Analysis (EDA)

5. Train-Test Split
6. Feature Scaling
7. Logistic Regression Modeling
8. Random Forest Modeling
9. Hyperparameter Tuning using Optuna
10. MLflow Experiment Tracking
11. Model Evaluation
